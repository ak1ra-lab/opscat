# opscat

## quick start

使用 `pip` 安装 `opscat`.

> 暂未上传至 PyPI, 需要下载到本地后执行 `pip` 命令安装.

```shell
git clone https://github.com/ak1ra-lab/opscat.git && cd optcat/
python3 -m pip --update pip && pip install .
```
